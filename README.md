## Run and build

- run in development mode
  ```bash 
  pnpm i
  pnpm electron:serve // run in Windows
  pnpm electron:servemac // run in mac
  ```
- build
  ```bash
  yarn
  yarn electron:build // build for Windows
  yarn electron:buildmac // build for mac 
  ```
