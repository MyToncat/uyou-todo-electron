English | [中文简体](https://github.com/tonylu110/uyou-todo-electron/blob/main/README/zh_cn.md)

## Run and build

- run in development mode
    ``` 
    yarn
    yarn electron:server
    ```
- build
  ```
  yarn
  yarn electron:build
  ```
